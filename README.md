# Excel Project Repository

This repository contains a collection of Excel projects I’ve completed to practice data analysis, dashboard creation, and data visualization. Each project showcases different Excel skills, including formulas, pivot tables, charts, and interactive dashboards.

Projects are designed to demonstrate practical applications of Excel for data analysis and decision-making.

## 🚀 Project 1 – Data Jobs Salary Dashboard

This project is an interactive salary dashboard for data-related jobs. It helps users explore how job title, location, platform, and employment type influence salaries.

<p align="center"> <img src="project_1_salary_dashboard/assets/Salary_Dashboard_Demo.gif" width="700"> </p>

Key Features:

- Compare median salaries across job titles (bar chart)
- Visualise salaries by country and job platform (map chart)
- Explore salary differences by job type (full-time, part-time)
